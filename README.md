# Beyond-the-Code-Who-Developers-Are-and-What-Drives-Them
This BA775 project focused on applying business analytics techniques to solve a real-world problem through data exploration, querying, visualization, and insight generation. As part of the project, we used SQL in Google Cloud Console to extract, clean, and analyze structured data, which helped in identifying key trends, patterns, and performance indicators. The technical work was combined with a business-oriented mindset to ensure that the analysis was not limited to numbers alone, but also contributed to meaningful and actionable recommendations.

To present the findings effectively, we used Tableau to build interactive dashboards and visualizations that translated raw data into a more intuitive and decision-friendly format. This helped in communicating insights clearly, highlighting important metrics, and making the results accessible from both an analytical and managerial perspective. The project strengthened our ability to work across the full analytics workflow — from querying and data preparation to visualization and business interpretation.

Overall, this project reflects our hands-on experience in business analytics, SQL, cloud-based data handling, and dashboard storytelling. It demonstrates our ability to use tools such as Google Cloud Console and Tableau to transform data into insights that support informed decision-making. More importantly, it showcases how weapproach analytical projects with both technical rigor and business relevance, which is a key skill wecontinue to build through BA775.
